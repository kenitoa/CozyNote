# 업데이트 예정 아이디어

## 실행 및 배포

- `CozyNote.cmd` 실행 시 JDK, Maven, JavaFX 준비 상태를 한 화면에 요약해서 보여주는 설치 진단 UI를 추가한다.
- Maven이 없는 환경에서도 안정적으로 실행되도록 Gradle Wrapper 또는 앱 전용 런타임 패키지를 포함하는 방식을 검토한다.
- 루트 구조를 `CozyNote.cmd` 하나와 앱 폴더 하나로 유지하는 패키징 스크립트를 추가한다.

## 화면 및 레이아웃

- 첫 실행 시 화면 크기를 감지해 `전체 패널`, `편집 집중`, `작은 화면` 같은 레이아웃 프리셋을 제공한다.
- 오른쪽 패널, 사이드바, 편집기 최소 폭을 설정 화면에서 조정할 수 있게 한다.
- UI 변경 후 스크린샷 기반으로 겹침, 잘림, 색상 대비를 확인하는 간단한 시각 검증 절차를 만든다.

## 음악 플레이어

- MP3 플레이어 UI에 CozyNote 테마 색상 프리셋을 추가한다.
- 앨범 이미지가 없을 때 사용할 기본 커버 아트를 CozyNote 스타일로 제작한다.
- 음악 플레이어에 현재 곡 목록, 즐겨찾기, 반복 모드 상태를 더 명확하게 표시한다.
- 파형 장식을 실제 재생 상태나 볼륨에 따라 움직이는 시각 요소로 발전시킨다.

## 개발 로그

- 작업 완료 시 날짜별 로그, 업데이트 예정, 위험 요소 문서를 자동으로 갱신하는 간단한 기록 명령을 만든다.
